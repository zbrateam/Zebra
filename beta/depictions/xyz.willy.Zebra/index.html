Zebra

  • Supports iOS 9.0 to 13.5

Description

  • Zebra is a package manager for jailbroken iOS devices.

    It supports most modern iOS versions and provides most features that Cydia does, while being notably quicker.

    If there are any issues you encounter, features you want to request, or anything else, please send me an email and I'll quickly get back to you.
